ROS driver node for HOKUYO 3D LIDARs
====================

This package provides a ROS driver node for 3D LIDARs with VSSP protocol.
VSSP protocol 1.0, 1.1 (for HOKUYO YVT-X002) and 2.1 (for HOKUYO YVT-35LX) are supported.
